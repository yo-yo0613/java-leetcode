# LeetCode Solutions in Java

This repository contains solutions to various LeetCode problems implemented in Java. Each problem includes the solution code and a detailed README with explanations.

## Problems Solved

### Array and HashMap
- [217. Contains Duplicate](ArrayAndHashMap/ContainsDuplicate/ContainsDuplicate.java) - Check if an array contains duplicate elements using HashSet.
- [169. Majority Element](ArrayAndHashMap/MajorityElement/MajorityElement.java) - Find the majority element that appears more than n/2 times.
- [1. Two Sum](ArrayAndHashMap/TwoSum/TwoSum.java) - Find two numbers in an array that add up to a target sum using HashMap.

### String
- [14. Longest Common Prefix](String/LongestCommonPrefix/LongestCommonPrefix.java) - Find the longest common prefix string amongst an array of strings.
- [125. Valid Palindrome](String/VaildPalindrome/ValidPalindrome.java) - Determine if a string is a valid palindrome, ignoring case and non-alphanumeric characters.

## How to Run
Each Java file contains a `main` method for testing. Compile and run using:
```
javac ProblemName.java
java ProblemName
```
